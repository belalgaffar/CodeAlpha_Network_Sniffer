# CodeAlpha_Network_Sniffer

## Overview
This repository contains a basic network packet sniffer developed as part of the Cybersecurity Internship at CodeAlpha. 

## Features
* Built using Python and the scapy library.
* Captures network traffic and extracts Source/Destination IP addresses.
* Identifies protocols (TCP, UDP, ICMP) and displays the payload.
